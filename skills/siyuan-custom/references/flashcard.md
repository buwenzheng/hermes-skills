# 闪卡系统指南

## 概述

思源笔记闪卡功能使用 **FSRS**（Free Spaced Repetition Scheduler）算法进行间隔重复学习。

FSRS 是一种先进的间隔重复算法，能够根据记忆表现智能调整复习间隔。

- GitHub：[open-spaced-repetition/free-spaced-repetition-scheduler](https://github.com/open-spaced-repetition/free-spaced-repetition-scheduler)

## 制卡规则

### 标记元素（挖空）

内容块中的 `==标记==` 元素会被视为挖空的答案。

```markdown
中国的首都是 ==北京==。
```

复习时，"北京"会被隐藏作为答案。

### 超级块

将超级块设置为闪卡时：
- **第一个子块** → 问题
- **其余子块** → 答案

### 列表/列表项

将列表或列表项设置为闪卡时：
- **非子列表内容** → 问题
- **子列表内容** → 答案

```markdown
- 什么是内容块？
  - 思源笔记的基本单位
  - 每个块有唯一 ID
```

### 标题块

将标题块设置为闪卡时：
- **标题** → 问题
- **标题下方的块** → 答案

## 使用方式

### 基于文档（推荐）

**优点**：
- 不必关注卡包
- 支持父子文档复习
- 更灵活

**操作步骤**：

1. **制卡**
   - 块标菜单 → 快速制卡
   - 快捷键：`Ctrl+Alt+F` / `⌥⌘F`

2. **复习**
   - 文档树面板右击文档 → 闪卡
   - 编辑器中点击文档块标 → 闪卡
   - 快捷键：`Alt+F` / `⌥F`

3. **全局复习**
   - 快捷键：`Alt+0` / `⌃0` 打开闪卡间隔重复面板

### 基于卡包（不推荐）

> ⚠️ 官方建议不要基于卡包使用闪卡，因为卡包是固定的，不如基于文档灵活。

如需使用：
1. 在 设置 → 闪卡 → 卡包 中开启
2. 块标菜单 → 添加到卡包...
3. 顶栏菜单 → 闪卡 进入复习

## 自定义属性

在文档块上设置以下自定义属性可覆盖全局设置：

| 属性名 | 说明 |
|--------|------|
| `riff-new-card-limit` | 新卡上限 |
| `riff-review-card-limit` | 旧卡上限 |

**注意**：不同文档上单独配置，相互独立。父文档配置为 5，子文档配置为 10，从父文档复习时按 5 计算。

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Alt+0` / `⌃0` | 打开闪卡间隔重复面板 |
| `Alt+F` / `⌥F` | 当前文档/选中文档间隔重复 |
| `Ctrl+Alt+F` / `⌥⌘F` | 快速制卡 |

## 复习评分

复习时可选择：
- **重来**：完全忘记，重新开始
- **困难**：记得但很费力
- **良好**：记得
- **简单**：轻松记得

FSRS 算法会根据评分调整下次复习时间。

## 注意事项

- 内容块被删除或 ID 改变后，对应的闪卡会失效
- 卡包数据存储在 `工作空间/data/storage/riff/` 目录下
- 使用 MessagePack 序列化协议存储

## 制卡设置

在 **设置 → 闪卡** 中可调整：
- 新卡上限
- 旧卡上限
- 制卡规则开关
